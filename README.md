# enika.vim
A Vim port of https://github.com/enkia/enki-theme

For better experience:
   - `set termguicolors`
   - `syntax on`
---

# Installation

## With a plugin manager

just add `Plug 'SpookySec/vim-enkia'` to the plugins
Add this to your `.vimrc` (or init.vim):

```vim
" For Vim 8.0 and above
color enkia
```
